# Healthcare Backend Setup

## التشغيل السريع:

1. **تنزيل المكتبات:**
\`\`\`bash
cd backend
npm install
\`\`\`

2. **تشغيل MongoDB:**
\`\`\`bash
# تأكد أن MongoDB يعمل على جهازك
mongod
\`\`\`

3. **إنشاء البيانات التجريبية:**
\`\`\`bash
npm run seed
\`\`\`

4. **تشغيل السيرفر:**
\`\`\`bash
npm run dev
\`\`\`

## حسابات تجريبية:
- **مريض:** patient1 / password123
- **طبيب:** doctor1 / password123  
- **مالية:** finance1 / password123

السيرفر سيعمل على: http://localhost:5000
